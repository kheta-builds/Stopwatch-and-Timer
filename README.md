# Timer & Stopwatch Application
A Java Swing desktop application built during my first year of Computer Engineering. It combines multiple time management tools into a single interface.

## Overview
This project was my introduction to Java and GUI development. It focuses on learning object-oriented programming, event-driven programming, and basic UI design.

## Features
### Stopwatch
- Start, stop, resume, and reset
- Lap recording functionality
- Displays elapsed time and lap times

### Countdown Timer
- Set custom hours, minutes, and seconds
- Pause and resume support
- Alert when timer reaches zero

### Preset Timer
- Save named timers (e.g., Study, Workout)
- Reuse saved timer configurations

### Pomodoro Timer
- 25-minute focus sessions
- Short break (5 min) and long break (10 min)
- Easy switching between modes

### Navigation
- Simple menu system to switch between tools
- Back navigation from all screens

## Technologies Used
- Java  
- Swing (JFrame, JPanel, JButton, JLabel)  
- javax.swing.Timer (event-driven updates)  
- NetBeans IDE (GUI Builder)  
- Absolute positioning (NetBeans-generated layout)

## How to Run
1. Open the project in NetBeans (recommended)
2. Run `MenuPage.java`

OR via terminal:
javac src/*.java
java -cp src MenuPage
